# Data from Kolada.se in Gapminder format

There is no bulk download of data, so this dataset was scraped from the API endpoints https://github.com/Hypergene/kolada in autumn 2024  

If you need to do it again / update the data, see instructions in https://observablehq.com/@open-numbers/kolada-se (access for open-numbers team only) and ETL scripts in this repo
